System requirements

  • Windows 10 or Windows 11 (64-bit)
  • x64-based processor (Intel / AMD)
  • Minimum 2 GB RAM (4 GB or more recommended)
  • Approximately 200 MB free disk space (up to 1 GB for future updates & data)
  • .NET 8 runtime included (self-contained executable)
  • No internet connection required after installation
  • Runs fully offline with local data storage only
  • Optimized for low CPU and memory usage
