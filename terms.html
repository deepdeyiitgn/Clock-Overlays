2. Description of Service

Transparent Clock is a free, offline*-first desktop clock overlay and focus companion application designed for Windows 10/11. The Software provides the following features:

  • Clock Overlay: An always-on-top transparent clock display
  • Pomodoro Timer: Focus session management with customizable work/break cycles
  • To-Do Manager: Local task management and organization
  • Focus Insights: Analytics and tracking of study/work sessions
  • Dashboard: Centralized view of all productivity tools
  • Utilities (Online): QR code generation/scanning and URL shortening (requires internet connection)

* The core functionality operates offline. Only specific utilities require internet connectivity.
